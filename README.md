# My Personal Portfolio

Welcome to the source code for my personal portfolio! This website serves as a showcase for my projects, technical skills, and professional journey.

## 🚀 About the Project
This portfolio is a custom-built, responsive website designed to provide a "cyber-sleek" user experience. It highlights my background in software development, including my work with C#, .NET, and various automation tools.

## 🛠 Tech Stack
Built using modern web technologies:
* **Frontend:** HTML5, CSS3 (with Custom Properties/Variables), and Vanilla JavaScript (ES6+).
* **Interactivity:** DOM API, Intersection Observer for animations, and LocalStorage for theme state management.
* **Responsive Design:** Mobile-first approach with CSS Grid and Flexbox.

## 📂 Project Highlights
* **IdentityFlow Validator:** High-performance C# automation engine.
* **Customer-Asset-Management-System:** Enterprise-grade REST API architecture.
* **Multithreading Excel Program:** Optimized data processing tool.
* **Learning Journey:** A collection of fundamental algorithmic and data structure implementations.
